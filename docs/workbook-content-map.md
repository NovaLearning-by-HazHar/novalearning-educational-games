# NovaLearning Workbook — 50-Page Content Map

> **Version:** 1.0 | **Last Updated:** 2026-02-09

---

## Articulation Hierarchy (Applied Throughout)

8-level progressive development framework. Each page targets one or more levels.

| Level | Name | Description | Typical Pages |
|-------|------|-------------|---------------|
| 1 | **Discrimination** | Visual/audio pattern recognition | 1-6, 33-34 |
| 2 | **Isolation** | Individual sounds/elements | 7-14 |
| 3 | **Syllables** | Sound pattern building | 15-20 |
| 4 | **Words** | Complete vocabulary development | 21-26 |
| 5 | **Sentences** | Structured communication | 27-32 |
| 6 | **Stories** | Narrative development | 43-46 |
| 7 | **Conversation** | Interactive dialogue | 47-48 |
| 8 | **Generalization** | Cross-context application | 49-50 |

---

## Template Types (from SA Workbook Design Guide)

| Code | Template | Description |
|------|----------|-------------|
| **ACT** | Activity | Multi-step interactive exercise |
| **COL** | Coloring | Large outlined images for crayon/pencil |
| **WRT** | Writing Practice | Letter/number tracing with guidelines |
| **MAT** | Matching/Connect | Draw lines between related items |
| **SEQ** | Sequencing | Order events/items in correct sequence |

## QR Code Scope (MVP)

Only **2 pages** have functional QR codes linking to web games:

| Page | Game | URL Path |
|------|------|----------|
| 15 | Count to 5 with Sipho | `/games/count-to-five` |

## Production Notes

- **Physical specs:** A4 portrait, 150gsm paper, perfect bound, laminated cover, 52 total (50 content + 2 covers)
- **Print:** 300 DPI, CMYK (Ghostscript conversion from Canva RGB exports), PDF/X-1a compliant
- **QR codes:** High-contrast black on white, 20×20mm, bottom-right corner, pages 15 and 22 only
- **No reading required:** All activities use picture prompts. Text is for parent/teacher reference only.
- **Cultural rotation:** Sipho, Thandi, Lerato appear most frequently (MVP characters). Pieter, Fatima, Amahle appear in group scenes and page 38.
